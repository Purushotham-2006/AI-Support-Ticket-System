# Support Ticket Classification & Prioritization System

## Overview

This project uses Machine Learning to automatically classify customer support tickets and assign priority levels.

The system helps support teams:

- Reduce manual sorting
- Detect urgent issues quickly
- Improve response times
- Generate support analytics

## Features

- Ticket Classification
- Priority Prediction
- Critical Ticket Detection
- Dashboard Analytics
- Category Distribution Chart
- Priority Distribution Chart
- Ticket Trend Analysis
- Search Functionality
- Report Download
- Dark Mode
- Recent Activity Table

## Technologies Used

- Python
- Flask
- Pandas
- Scikit-Learn
- TF-IDF Vectorization
- Multinomial Naive Bayes
- HTML
- CSS

## Machine Learning Workflow

1. Data Collection
2. Text Preprocessing
3. TF-IDF Feature Extraction
4. Model Training
5. Ticket Classification
6. Priority Prediction
7. Dashboard Visualization

## Run Project

```bash
pip install -r requirements.txt
python app.py
```
